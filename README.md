# Limpiezas G3

Landing page profesional para Limpiezas G3, optimizada para SEO local en Álava, Amurrio, Llodio, Orduña, Bilbao y Vitoria-Gasteiz.

## Publicación

Este repositorio incluye un workflow de GitHub Pages en `.github/workflows/pages.yml`.

Para publicarlo:

1. Sube todos los archivos al branch `main`.
2. En GitHub, ve a `Settings > Pages`.
3. En `Build and deployment`, selecciona `GitHub Actions`.
4. Ejecuta el workflow `Deploy static site to GitHub Pages` si no arranca automáticamente.
